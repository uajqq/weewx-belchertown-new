# New Belchertown WeeWX skin

### See the skin in action live [here!](https://www.uajqq.com/new-belchertown)

This skin is a fork of [Pat O'Brien](https://github.com/poblabs)'s excellent Belchertown skin for the [WeeWX weather software](http://weewx.com) and is modeled after [BelchertownWeather.com](https://belchertownweather.com). 

Features include:
* Modern, responsive card-based layoutd
* Real-time streaming updates
* Extensive charting system with full customized control on observations, historical timescale, grouping and more
* Light and dark mode support
* Forecast data
* Almanac (sun/moon) data
* Information on your closest earthquake
* Weather records -- day, week, month, year, all time
* Special views for Kiosk mode, and Raspberry Pi displays
* Many supported languages, allowing automatic translation
* All information updates automatically, so page refreshes are not necessary


## Documentation, installation, configuration

Full documentation on installation, configuration, the skin's extensive features, and troubleshooting, can be found in the [wiki](https://github.com/uajqq/weewx-belchertown-new/wiki).

If you are upgrading from a pre-2.0 version, make sure you read the installation guide carefully!
